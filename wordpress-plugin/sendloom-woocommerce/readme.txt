=== Sendloom Connect ===
Requires at least: 6.4
Tested up to: 6.8
Requires PHP: 8.0
Stable tag: 3.0.0
License: GPLv2 or later

Connects a WooCommerce store to Sendloom: syncs customers, orders and products,
and streams behavioural events (product views, searches, cart activity,
checkouts, purchases) for email marketing and growth intelligence.

== Installation ==

1. Copy this folder (wordpress-plugin/sendloom) into wp-content/plugins/.
2. Activate "Sendloom Connect" in WordPress admin (WooCommerce must be active).
3. Open WooCommerce → Sendloom.
4. Enter your Sendloom URL and the store API key
   (Sendloom → Settings → API & webhooks).
5. Save & connect, then press "Sync now" for the initial back-fill.

Full local walkthrough: see PLUGIN.md in the Sendloom repository.
